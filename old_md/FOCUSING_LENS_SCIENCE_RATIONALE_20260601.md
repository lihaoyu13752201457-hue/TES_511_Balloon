# 聚焦镜能段与科学论证(20260601)

> 时序标记 **20260601**。本文回答一个核心设计问题:**Laue 聚焦镜到底该聚焦 511 线,还是连续的 480–550 keV?**
> 结合优先文献 **511-CAM (arXiv:2206.14652)** + 正电子湮灭谱物理 + CLAIRE,给出论证与设计取向。
> 用途:供 Codex 据此修正聚焦镜设计(配合 `LAUE_LENS_DESIGN_SPEC_20260601.md`)。
> 结论先行(TL;DR):**光学聚焦"511 线";480–550 是"探测器/分析窗"(线轮廓+两侧基线),不是光学要去聚焦的范围。
> 焦距用 511-CAM 自己的 Laue 数 ~8.3m,不是 35m。sub-511 的 o-Ps 连续谱不是 511-CAM 科学,留作卫星未来展望。**

---

## 1. 优先文献 511-CAM 实际怎么说(arXiv:2206.14652, JATIS 2023,已核全文)

- **科学纯粹是 511 线**:发现/定位点源(亚角分)+ **测线的视线速度与 Doppler/引力展宽**(390 eV 分辨率就是为了分辨线的展宽与移动)。
- **光学**:Channel 基线 = **12m / 50.89 cm²**(即主链 Step07 那个占位标量来源);**Laue 备选 = 8.3m 焦距**(论文明确"从 CLAIRE 2.77m@170keV 缩放"),概念上可到 200m(卫星编队)。
- **全文没有 ortho-positronium 连续谱 / positronium fraction / sub-511 连续谱**——511-CAM 是**纯线仪器**。
- **平台**:明说适合 (U)LDB 气球**或卫星**。

## 2. 正电子湮灭谱物理(决定能段该怎么取)

- **511 keV 线**(2γ):窄+宽分量;**会被 Doppler/引力展宽、被视线速度移动**——这正是 511-CAM 要测的量。被展宽/蓝移的线**会跨过 511 延伸到上方**。
- **ortho-positronium 连续谱**(3γ):**全部在 511 以下**;480–511 是它最靠近线、最有诊断价值的上沿。线/连续比 = **positronium fraction**(银心 ≈1.08)→ 约束 ISM 温度/电离。
- **511 以上**:无正电子湮灭信号本体,但**有用**——是被展宽线的高翼 + **干净的连续谱基线**(定线流量要两侧基线)。

## 3. 论证:两个常见误解的澄清

### 误解A:"上线应该正好卡在 511"
**错。上线要比 511 高一点。** 因为 511-CAM 的核心科学就是测**线的展宽与移动**:被展宽/蓝移的线延伸到 511 以上,且定线流量需要 511 两侧的连续谱基线。所以**分析能段必须跨过 511 往上**(~511 + 一二十 keV 的量级;名义到 550 作为基线余量也可接受)。

### 误解B:"480–550 都得让镜子去聚焦"
**错,而且是最关键的区分:**

> **光学聚焦的是"线(511)";480–550 是"探测器 / 分析窗"(线轮廓 + 两侧连续谱基线),不是"光学要去聚焦的能段"。**

- Laue 镜每个半径只聚焦一个能量;把晶体摊到 480 和 550,会**稀释 A_eff(511)**(线灵敏度下降),而 480/550 那两段本来只需当**基线**用。
- 基线(480–511、511–550)由 **TES 谱直接测**即可(基线不需要高有效面积),**不需要镜子去聚焦**。
- 所以:**镜子把 A_eff 堆在 511(及紧邻线轮廓 ~511±几keV);480–550 留给探测器做谱分析。** 两者各司其职。

## 4. sub-511 连续谱(o-Ps)要不要做?能不能卫星做?

- **511-CAM 没把它列为科学**(纯线)。所以下探 o-Ps 连续谱是**超出 511-CAM 既定科学的加项**。
- 物理上 o-Ps 连续谱是真科学(positronium fraction → ISM),但它**又宽又弱**;**聚焦望远镜不是测连续谱的好工具**(连续谱是 SPI 类无聚焦谱仪的强项,聚焦镜强在线)。
- **卫星升级**:论文提到概念上可到 200m 大镜。卫星大镜**理论上**可把上沿 o-Ps 连续谱(480–511)也聚焦、胜过无聚焦谱仪 → 是**合理的"卫星 future prospects"故事**,但**不是聚焦镜的天然主场,也不是现在气球的核心科学**。

## 5. 焦距:用 511-CAM 自己的 Laue 数 ~8.3m,不是 35m

- **511-CAM 的 Laue 概念 = 8.3m**(从 CLAIRE 2.77m 缩放)。当前 demo 原本的 8.3m 其实就是这个数;Codex 改成的 **35m 偏离了 511-CAM**(更大面积但不是它的数)。
- 短焦(8.3m)→ 环半径小(r(511)≈62mm)→ 单环面积小;**正解是像 CLAIRE 那样多环密排 + 满方位填充**来堆 A_eff,**不是拉长焦距**。
- 若确实要更长焦(更大 A_eff),那是**走"卫星 / 编队"故事**(论文的 200m 概念区间),必须单独论证机械可行性(气球给不了 35m 刚性结构),不能默认。

## 6. 结论与设计取向(给 Codex)

| 层次 | 取值 | 说明 |
|---|---|---|
| **科学核心** | 银心 **511 线**:定位 + 速度/展宽 | 完全对齐 511-CAM,纯线 |
| **光学聚焦** | **聚焦 511**,f ≈ **8.3m**,A_eff 堆在 511 | 511-CAM Laue 数;晶体密簇在 511 线区 + 满方位密排 |
| **探测器/分析窗** | **~480–550 keV**(线轮廓 + 两侧基线) | TES 直接测谱,镜子不需聚焦整带 |
| **卫星 future prospects(可选)** | 大镜下探 480–511 o-Ps 连续谱 → positronium fraction | 写进论文展望,不当现在主科学 |

**给设计脚本的具体修正**:
1. 焦距回 **~8.3m**(除非明确走卫星故事并论证)。
2. **晶体堆在 511 线区**(满方位密排 + 线轮廓所需少数几环),最大化 A_eff(511);**不要**为名义"480–550"把环平摊到 480/550。
3. **480–550 作为 TES 分析窗**写进探测器/Step08,**不**写成光学聚焦要求。
4. 论文结构:核心 = 511 线灵敏度(气球);**future prospects 一段** = 卫星大镜测 o-Ps 连续谱/positronium fraction。

## 7. 当前执行口径

本轮按**只做 511 线核心科学**执行,**定稿设计见 `EXECUTE_ROUTE_A_FULLCHAIN_AND_REPORT_20260601.md` §2**:
Ge(111) 单环 511、f=9m、30″ 镶嵌、A_eff(511)≈16cm²、天然能窗(FWHM)**≈501–521keV**。
统计宽窗 W1 用这个**设计天然能窗 ~501–521keV**(不再用 480–550);最终 3σ 用 511±420eV;参考源 V404 511 扫强度。
positronium fraction / o-Ps 连续谱不作为当前气球光学设计目标,只保留在卫星大镜或后续扩展的展望里。

如果未来项目口径明确要把 o-Ps 连续谱纳入主科学,再另开宽带光学/非聚焦谱学方案;不要把这个未执行目标混进当前 511 线镜的 A_eff 归一化。
