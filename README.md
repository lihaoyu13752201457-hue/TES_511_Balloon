# TES_511_Balloon

Detector-coupled simulation workspace for a balloon-borne focused 511 keV TES Laue-lens telescope.

Current paper-facing entry points:

- `core_md/README.md`: project memory index and current authority boundaries.
- `core_md/workflow.md`: stepwise workflow, commands, and interpretation boundaries.
- `Bgo_sample/README.md`: current equal-attenuation BGO active-shield material branch.
- `Bgo_sample/CLOSURE_SUMMARY.md`: tracked BGO Step05--Step08 and BGO-vs-CsI closure digest.
- `Bgo_sample/EXTENDED_CLOSURE_SUMMARY.md`: tracked BGO spatial, threshold-replay, and material-attenuation sidecar closure.
- `core_md/balloon511_nima_latex_drafts/`: English and Chinese NIM A manuscript drafts.

Large simulation products under `runs/`, `outputs/reports/`, and `stepwise_maintenance/**/outputs*/` are intentionally ignored unless explicitly promoted to tracked paper-facing summaries.
