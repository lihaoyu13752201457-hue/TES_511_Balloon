# Paper Source Figure/Table Assets

This directory holds manuscript-facing figure and table assets that are copied
from validated simulation outputs or generated specifically for the LaTeX
drafts.

Current mass-model figures:

- `fig_mass_model_xz_overview.png`: full X--Z detector/cryostat overview copied
  from the current geometry-closure render output.
- `fig_mass_model_detector_bay_detail.png`: detector-bay detail copied from the
  current geometry-closure render output.

Current optics figures:

- `fig_optics_laue_transport_schematic.png`: radial--z Laue transport
  visualization generated from the current 10 m Ge(111) ring configuration and
  tracked accepted focal-plane crossings.
- `fig_optics_focal_crossings_xy.png`: transverse focal-plane crossing
  distribution accepted by the Be-window gate and used for detector replay.
- `fig_optics_generation_metadata.json`: input/output provenance and focal-spot
  summary for the optics figure generation.

Current detector-selection and time-axis figures:

- `fig_selection_veto_time_axis.png`: four-panel diagnostic generated from the
  current detector-selection, mission-time, and significance products for the
  M=50000 sampled-position chain. It
  shows line-window raw/active-veto/Compton-FoV rates, survival fractions, Compton/FoV
  class counts, and the mission-time fold with accidental live factor.
- `fig_compton_fov_geometry.png`: side-entry Compton/FoV geometry schematic
  generated from the selection Be-window disk center, normal, and radius, with the
  Compton kinematic equation used by the selection code.
- `fig_selection_veto_time_axis_summary.json`: machine-readable provenance and
  key values for the detector-selection figures.

Source note:

- The source of truth is the geometry-closure output, not this copied directory.
- These figures are paper-facing render assets for the current detector/cryostat
  proxy mass model.
- If the geometry is regenerated, refresh these copies from the geometry render
  output directory and recompile the LaTeX drafts.
- The optics figures are generated by `build_optics_figures.py` from the current
  10 m Ge(111) configuration and its tracked diffracted focal-plane
  crossings. If the optics inputs are regenerated, rerun that script and recompile
  the LaTeX drafts.
- The detector-selection figures are generated by
  `build_selection_time_axis_figure.py` from the current M=50000
  detector-response summary, background time-variation CSV, and
  cumulative-significance CSV. If these products are regenerated, rerun that script
  and recompile the LaTeX drafts.
