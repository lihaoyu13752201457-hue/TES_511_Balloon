## 0. 一句话结论

prompt-511 偏高**是真实几何效应, 不是归一化假象**。根因不是"窗太薄漏光", 也不是
"开口直通", 而是: **45° 侧入设计把老链路那根厚侧屏蔽柱 (CsI 4cm+Cu+钢, 48–81 g/cm²)
换成了同轴的薄铝低温杜瓦壳 (Still/4K/60K/真空夹套, 每层 1–4 mm Al, 半径 8.5→13.3 cm)。**
大气 e⁺ 在这些近探测器铝壳里湮灭出 511, 沿径向向内打到近轴 TES。

**怎么办 — 修正后的策略:**
- **不要把焦斑 ROI 当成主压本底策略。** `spot_r90` 能作为分析诊断显示
  prompt-511 在 TES 面上比信号弥散, 但它不能解释旧 `new_geo_re` hard-window prompt 为何较低,
  也不能修复当前侧口/侧壁产生 prompt 的硬件根因。ROI 相关结果只保留为 sidecar/diagnostic,
  不再作为推荐设计或论文头条口径。
- **主线应回到本地侧口/侧壁几何修复。** 杜瓦壁内侧 shadow shield / graded-Z collar /
  CsI collar / liner-continuity 这类局部方案仍需 MC+CAD 验证。解析筛选给出 eplus
  `0.0543 -> ~0.011 cps` 的上界, 但必须确认 W 自发射 add-back、延迟活化、overlap 和信号锥裁剪。
- **不要**: 侧口孔径挡环 (1.4×, 无效); 无差别全局加质量 (抬高延迟活化)。

可复现脚本与输出已落盘 (`outputs/reports/prompt511_entry_audit_20260617/build_prompt511_fix_screen.py`
→ `prompt511_fix_screen_summary.json`); ~5×/Z→11/1Ms→3.5e-5 是**解析上界, 未 MC, 不得作权威引用**。

## 1. 根因 (逐粒子 + 几何, 全部本人复算)

### 1.1 是真实有效面积上升, 不是 bug, 也不是"纯归一化" (措辞较上版收敛)

Codex reviewer 的纠正成立, 我把上版"半径是 red herring"的过满说法收敛如下:
绝对 cps 比 = **2.220 = 每事件权重(面积)比 2.94 × 单粒子选择效率比 0.755**。即 cps 的算术里, 面积
(远场/包络面) 因子确实在; 但这**不是归一化 bug/假象**, 而是更大包络面下的**物理有效面积**表达。
两边各自自洽 (我核实 v3p5 `SurroundingSphere 60`, N_gen=243727 → 每 rep TT=184.2 s 与盘上 TT 逐位
吻合 80/1472.3=0.0543377 ✓; 老 R=35 反算 106/4331=0.024474 ✓; A_eff=N_sel/N_gen×Area 对球半径不变)。
**两条都要说清, 否则会误读**: ① 真实不变量 A_eff(大气 e⁺→W2 511) = 0.209→0.465 cm² (×2.22) 确是物理翻倍;
② 但**不能**把 ×2.22 读成"新几何选择效率更高"——按单位投掷粒子算, 效率反而**降到 0.755**。物理图像:
更大的仪器 (包络半径 49.6 vs ~30) 截获更多大气通量, 净有效面积升 2.22×。

### 1.2 入射统计: 窗 vs 非窗 (两链路)

| | 新 v3p5 (80 事件) | 老 new_geo_re (106 事件) |
|---|---|---|
| 严格穿光学窗 (Be+Al 盘 r<1.898) | **1/80** | — (老窗在顶部轴向) |
| **非窗** | **77/80 (96%)** | — |
| 511 出生体积 Top1 | **侧口侧壁 61/80** | **Outer_Al_Mech_Shell 88/106 (顶/轴向)** |

设计光学窗几乎不漏 (1/80); 511 主要来自 `Vacuum_Jacket_Al_..._side_port_side_wall` /
`Outer_Al_Mechanical_Shell_..._side_wall` / `Passive_W/Cu_Liner_..._above_side_port` 这些杜瓦壁结构。

### 1.3 侧向视线的 511 透射率 (固定弦, 511 keV μ/ρ×ρ×厚)

| 视线 | 面密度 g/cm² | 511 透射率 |
|---|---:|---:|
| 新·侧窗设计线 (Be 0.015+Al 0.013) | 0.06 | **99.5%** |
| 新·侧向中心线 | 3.6 | 73.8% |
| 老·对应外→内侧线 (CsI 4+Cu 1.63+钢 1.17+Al 1.1+…) | 48.3 | **1.6%** |
| 老·侧向中心线 | 80.7 | **0.11%** |

同一侧向, 老几何放 48–81 g/cm² (透射 0.1–1.6%), 新几何只剩 0.06–3.6 g/cm² — 差 50–700 倍。

### 1.4 逐粒子跟踪 (材料/厚度 + 截止点)

- **新·进 TES 的选中 511**: 出生在侧口铝壁, 沿途仅沉积 0–92 keV 就把 511 给 TES。
  例: `id84231` born `Vacuum_Jacket_Al_..._side_port_side_wall` → 停 TP_L2 [TES], 途中 84 keV。
- **老·侧向 511 (未进 TES, 看它截止在哪)**: 67/100 停在 `CsI_Active_Shield_Side00–07` (沉积 ~520 keV 全能量)。
  例: `id12` 穿 CsI 10.7cm → 停 CsI_Active_Shield_Side00。**老几何把侧向 511 喂给侧 CsI 屏蔽。**
- **老·进 TES 的选中 511**: 88/106 出生在 `Outer_Al_Mech_Shell`, 走顶/轴向。

> 注意: 被选中的 511 并非从 0 材料窗口直冲进来 — 沿湮灭→TES 直线中位面密度新 20.7 / 老 31.2 g/cm²
> (透射 17% / 7%), 只有 1/80 走干净窗。它们是**穿过中等材料的"幸存光子"**, 源自侧壁那一大片铝发射体
> (立体角大), 不是一条空管子。这一点很关键, 决定了下面方案 A 为何失败、方案 B 为何要"面状"而非"孔状"。

## 2. 方案试做与量化 (本人对 80 事件做屏蔽筛选)

几何坐标已厘清: 杜瓦是绕**局部 z 轴**的同轴薄铝壳 (Still 8.5–8.6, 4K 9.9–10.0, 60K 11.4–11.6,
真空夹套 12.9–13.3 cm); 侧口是 z≈−5.2 处一个 **~28° 的 φ 扇区缺口** (绕 φ≈180°, 即 −x 看轴);
TES 在 r≈2–3.5 cm、z≈−5。信号只需这一个 φ 扇区开着。

### 方案 A — 侧口挡环 (aperture baffle): **失败 (1.4×)**
窗平面环形挡 W (r≥1.898, 厚 1–3 cm) → 残留 70–72% (1.4×)。
**原因: 80 个事件里只有 34 个穿窗平面, 46 个 (57% 的漏率, 0.031 cps) 根本不过窗平面** —
它们从杜瓦壁各个 φ/z 直接横进探测器舱。孔状挡环管不到面状漏。
**这也修正了 Codex "local side-port closure" 若理解为孔径挡板的说法 — 挡孔没用。**

### 方案 B — 杜瓦壁高-Z 内衬 (areal W liner): **有效 (~5×)**
绕 z 轴的 W 圆柱壳, 覆盖全 φ **除信号端口扇区 (φ≈160–200° 留空)**, 半径在 TES 与杜瓦壁之间:

| W 内衬 (绕 z 轴) | 残留 prompt-eplus | 压制 |
|---|---:|---:|
| r=[4,6] cm (2cm), 端口扇区开 | 0.0109 cps (20.1%) | **5.0×** |
| r=[5,7] cm (2cm), 端口扇区开 | 0.0145 cps (26.7%) | 3.8× |
| r=[5,8] cm (3cm), 端口扇区开 | 0.0139 cps (25.5%) | 3.9× |

79/80 事件出生于 r>5cm (壁内/壁外), 69/80 向内飞 — 故壁内 W 壳能拦下大多数。
**信号端口扇区留空 ⇒ 当前信号接收不受影响** (信号经该扇区聚焦到近轴 TES)。

### 端口同孔径地板: ~15%
0.00815 cps (15%) 的漏过事件穿过开着的端口扇区进来 — 与信号同孔径。**在保持信号端口开放的约束下,
被动屏蔽不能简单挡掉这部分**(除非重新设计信号/屏蔽共形结构或更精细准直)。ROI 可以诊断它在
TES 面上相对弥散, 但不应被写成压本底主策略。这个地板应作为局部侧口设计的残差/约束处理。

## 5. 状态与局限 (重要 — 不要越界引用)

- 本方案是**解析级几何屏蔽筛选**: 对真实 80 个事件的湮灭→TES 直线做 W 衰减 (μ_W=1.771/cm @511keV)。
  它正确捕捉**几何遮挡** (主效应), 但**未包含**:
  1. **W 内衬自身的 511 自发射 (add-back)** — e⁺ 在 W 内湮灭产生的新 511, 会**回补**一部分, 使实际压制 < 5×。
     **这是 ~5× 为"上界"的主因, 必须 MC 确认。**
  2. 被挡 511 的康普顿下散射回 W2 (W2 窗很窄, 影响小)。
  3. 信号经端口的精确保留率 (几何上端口开着→≈不变, 但内衬边缘贴近信号锥需 MC 校)。
- 故 §2/§3 的 ~5× 与 §3 的 Z→11 都是**解析上界, 未经 MC**。不可作为新权威/论文数字引用。
