# Legacy Archive

This directory holds assets that are useful for provenance but are not part of
the current fix5 geometry workspace.

## Layout

- `code/tools/`: old BGO, v3p5, prompt511, and new_geo_re helper scripts.
- `code/geometry/`: old DEMO2/v3p5 geometry builders.
- `config/`: old source-card templates and science/source configs.
- `data/`: small standalone smoke/isotope leftovers.
- `docs/core_md_legacy/`: previous project memory, old workflow, prompt511
  contracts, Claude review notes, and archived handoff docs.
- `docs/old_md/`: older planning and route notes.
- `docs/paper_draft/`: superseded paper skeleton material.
- `docs/reviews/`: top-level historical review files.
- `geometry/outputs/`: old generated geometries, including v2p2, baseline v3p5,
  rejected multiholeW variants, and pre-fix5 user redesigns.
- `geometry/references/`: old BGO sample branch, v3p5 reference assets, and
  geometry records.
- `reports/`: old report directories for v3p5, BGO, prompt511, and new_geo_re.
- `runs/`: small remaining old run sidecars that were not deleted as raw waste.
- `stepwise_maintenance/`: old stepwise outputs and old step-specific code/docs.
- `tests/`: legacy smoke and validation scripts.

Raw `.sim.gz`, `.dat`, `.log`, and other large generated files remain governed
by the repository ignore rules.
